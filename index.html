CORE
MODULES

// MODULE 01
👁️
Vision Detection Module
AI · COMPUTER VISION

An AI-based vision detection system designed to identify emergency situations from images and video data. It utilises advanced object detection models such as YOLO to analyse visual content in real time and detect critical events including fire, smoke, weapons, and abnormal human activities.

The system processes video frames sequentially, extracting important visual features and generating bounding boxes with confidence scores for detected objects. By automating visual monitoring, this module significantly reduces the need for continuous human supervision and enables faster identification of potential threats in surveillance environments.

// MODULE 02
🎙️
Audio Detection Module
SIGNAL PROCESSING · CNN

Focuses on analysing audio data to detect potentially dangerous events that may not be clearly visible in video footage. It uses signal processing techniques such as Mel-Frequency Cepstral Coefficients (MFCC) along with deep learning models like Convolutional Neural Networks (CNNs) to classify sounds.

The system is capable of identifying events such as gunshots, explosions, and accident-related noises. By integrating audio-based detection, this module enhances the overall reliability of the system, especially in scenarios with poor visibility or occluded video input, providing an additional layer of security.

// MODULE 03
🧠
Criminal Identification Module
FACE RECOGNITION · MULTI-FRAME

Responsible for identifying individuals using advanced face recognition techniques. It employs models such as ArcFace or FaceNet to extract unique facial features and compare them against stored records in a database. The system is designed to function effectively under real-world conditions, including partial occlusion, masks, and varying lighting environments.

To improve accuracy, multi-frame verification is used, where multiple frames are analysed before confirming an identity. This module plays a crucial role in supporting law enforcement and security operations by enabling fast and reliable suspect identification.

// MODULE 04
🚨
Violent Behaviour & Object Detection
CNN–LSTM · IOU OCCLUSION

An AI-powered real-time surveillance system designed to enhance public safety by automatically detecting violent behaviour and missing objects from video data. The system processes CCTV or uploaded video footage and applies advanced deep learning techniques to analyse both spatial and temporal patterns.

A CNN–LSTM-based model recognises suspicious or violent human activities such as fighting, assault, and robbery by understanding motion across consecutive video frames. A YOLOv8-based object detection module simultaneously monitors objects within the scene and identifies when an item has been removed. An IoU-based occlusion detection mechanism distinguishes between objects that are temporarily hidden and those that are actually stolen — reducing false alerts. All components are integrated into a unified pipeline supported by a Flask backend and a React-based web interface, allowing users to upload videos, view detection results, and receive real-time alerts.
