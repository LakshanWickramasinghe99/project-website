LITERATURE
SURVEY

Modern research in public safety systems focuses on Artificial Intelligence (AI), Computer Vision, and Audio Analysis to automate surveillance and detection tasks. Previous systems have explored a range of technologies — from YOLO-based detection to multimodal fusion — each contributing to the foundation CISS builds upon.
// EXISTING APPROACHES

What Systems Have Used

  • YOLO-based object detection for identifying fire, smoke, weapons, and abnormal activities
  • ArcFace and FaceNet face recognition models for criminal identification
  • Audio classification models to detect events like gunshots and accidents
  • Multimodal systems combining video and audio to improve accuracy
// LIMITATIONS OBSERVED

Where Existing Systems Fall Short

  • Focus on a single modality — only video or only audio
  • Depend heavily on manual monitoring
  • Lack secure data storage mechanisms
  • Studies confirm that combining multiple technologies improves both detection accuracy and system reliability

GAP

01
🔓

Insecure centralised databases — most systems store criminal records in centralised systems with no tamper-proof guarantees.

Sri Lankan context ignored — limited solutions are designed specifically for the operational conditions and needs of Sri Lanka.

03 / 05

RESEARCH
PROBLEM

// PROBLEM STATEMENT

Current Systems In Sri Lanka Face Critical Challenges

  • Manual video analysis is time-consuming and inefficient
  • AI models struggle with low-quality or occluded images
  • Criminal records in centralised systems are vulnerable to hacking
  • No unified system exists to process uploaded videos and evidence

Therefore, the core problem is: the lack of a secure, intelligent, and practical system to automatically analyse video/audio data, identify threats or suspects, and store results in a tamper-proof way.

SYSTEM
COMPONENTS

// COMPONENT 01

Vision Detection Module

This component is an AI-based vision detection system designed to identify emergency situations from images and video data. It utilizes advanced object detection models such as YOLO to analyze visual content in real time and detect critical events including fire, smoke, weapons, and abnormal human activities. The system processes video frames sequentially, extracting important visual features and generating bounding boxes with confidence scores. By automating visual monitoring, it reduces the need for continuous human supervision and enables faster threat detection.

// COMPONENT 02

Audio Detection Module

This component focuses on analyzing audio data to detect potentially dangerous events that may not be visible in video footage. It uses MFCC feature extraction and deep learning models such as CNNs to classify sounds like gunshots, explosions, and accident noises. By integrating audio-based detection, this module enhances system reliability in low-visibility environments and provides an additional layer of security.

// COMPONENT 03

Criminal Identification Module

This component is responsible for identifying individuals using advanced face recognition techniques such as ArcFace and FaceNet. It extracts unique facial features and compares them with stored records. The system works under real-world conditions including masks, occlusion, and varying lighting. Multi-frame verification improves accuracy and reduces false identification, supporting law enforcement operations.

// COMPONENT 04

AI Surveillance System

This component presents an AI-powered real-time surveillance system that detects violent behavior and missing objects from video data. It uses a CNN–LSTM model to analyze motion patterns and identify suspicious activities such as fighting and robbery. A YOLOv8-based module monitors objects and detects removal events, while an IoU-based mechanism distinguishes between occlusion and actual theft. All components are integrated into a unified pipeline using a Flask backend and React dashboard, enabling real-time alerts, video uploads, and interactive monitoring. This system reduces manual effort, improves response time, and provides a scalable solution for intelligent surveillance.
